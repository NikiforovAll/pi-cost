pi-cost

Cost dashboard for the pi coding agent. Reads JSONL session files from ~/.pi/agent/sessions.

Costs come from two sources:

actual — pi recorded the cost in the JSONL.
estimated — computed from token counts and LiteLLM pricing (refreshed every 6h).
